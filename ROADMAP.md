# Label Ease - 8 Week Development Roadmap

## Phase 0: Validation (Weeks 1-4)

### Week 1: CAB Recruitment
- [ ] Identify 5 target Etsy sellers
- [ ] Draft CAB pitch
- [ ] Send DMs + cold emails (Reddit, Facebook groups, Etsy forums)
- [ ] Prepare prototype mockups

### Weeks 2-3: CAB Interviews + Feedback
- [ ] 30-min interviews with 5 sellers
- [ ] Show Figma mockups
- [ ] Gauge pricing elasticity ($24, $49, $99)
- [ ] Document feedback

### Week 4: Go/No-Go Decision
- [ ] Analyze: Do 3+ sellers pre-commit?
- [ ] **Gate**: Proceed to MVP build or pivot

## Phase 1: MVP Build (Weeks 5-8)

### Weeks 5-6: Core Development
- [ ] Supabase schema + auth
- [ ] Label designer (React component)
- [ ] QR generation (backend + frontend)
- [ ] PDF export (pdfkit + Avery templates)

### Week 7: Mobile + Integration
- [ ] React Native app (Expo)
- [ ] QR scanner (camera)
- [ ] Real-time sync (Supabase subscriptions)
- [ ] Testing + polish

### Week 8: Alpha Launch
- [ ] Beta access for CAB
- [ ] Feedback iteration
- [ ] Prepare for soft launch

## Success Criteria

- ✅ 5 CAB sellers recruited + 3+ pre-commits
- ✅ MVP ships in 8 weeks
- ✅ 5 users scanning 500+ items/week
- ✅ Churn <5%, NPS >50
